# Implementation-of-Single-Source-Shortest-Path-Algorithm-Dijkstra
A Python and Streamlit application that implements Dijkstra's Single Source Shortest Path Algorithm to compute the shortest distance and path from a source vertex to all other vertices in a weighted graph.

## Overview
This project demonstrates the implementation of Dijkstra's Algorithm using Python and Streamlit.

The application computes the shortest path from a selected source vertex to every other vertex in a weighted graph.

## Features
Implementation of Dijkstra's Algorithm,
Displays shortest distance to every vertex,
Displays shortest path,
Easy-to-use Streamlit interface.

## Technologies Used
Python,
Streamlit.

## Algorithm
Dijkstra's Single Source Shortest Path Algorithm.
